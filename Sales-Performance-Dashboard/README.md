# Sales Performance Dashboard (Excel)

## Project Overview
This project is an Excel-based Sales Performance Dashboard created to analyze sales data and provide meaningful business insights using pivot tables, charts, and slicers.

## Dataset
The dataset contains sales transaction details such as:
- Sales Date
- Region
- Product
- Salesperson
- Units Sold
- Sales Amount
- Profit

## Key Metrics
- Total Sales
- Total Units Sold
- Total Profit
- Average Sales

## Analysis & Charts
- Total Sales by Region (Donut Chart)
- Total Sales by Product (2D Bar Chart)
- Total Sales by Salesperson (2D Column Chart)
- Units Sold by Product (Line Chart)

## Dashboard Features
- Interactive slicers for Region, Product, and Salesperson
- Dynamic charts connected to pivot tables
- Single-page dashboard for quick decision-making

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
